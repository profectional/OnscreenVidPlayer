# Video Player

This is a video player application built with PyQt5 and Python. It plays all .mp4 files in the same directory with 50% opacity on top of your other windows without interuption.

## Installation

1. Ensure you have Python installed on your system. If not, download and install it from the official Python website. During the installation, make sure to select the option to add Python to your system path.

2.Open a terminal and run the following commands:

python setup.py

or:

pip install PyQt5
pip install moviepy

install K-litekodec.exe

Running the Program:
To run the program double click it after all instalations or open a terminal in the directory/folder containing the start.py file and run the following command:

python start.py

Usage
Once the program is running, you can interact with it using the following keyboard commands:

Esc to exit the program
r to replay the current video
s to skip to the next video
p to go to the previous video
Up to increase the window opacity
Down to decrease the window opacity
w to make a new file and increase the volume permanently (renamed as x_louder)
e to decrease the volume (this setting is saved)
Space to pause/play the video
Adding Videos
You can add your own videos to the program by placing them in the same directory as the main.py file. The videos should be in .mp4 format.
